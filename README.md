# 100시간 공부 앱

뽀모도로로 **100시간 공부**를 채우고, 매일 공부한 시간과 성찰을 기록하는 웹앱입니다.
아이폰·아이패드에 설치해 앱처럼 쓸 수 있고, **모든 기록은 이 기기 안에만** 저장됩니다(서버 없음).
`ios-app/` 폴더를 이용하면 진짜 아이폰 앱으로도 설치할 수 있습니다 (→ `ios-app/README.md`).

## 주요 기능
- ⏱ **뽀모도로 타이머**: 5분 단위 빠른 선택(5~30분) + **±30초/±1분 미세 조절** (예: 1분 30초)
- ☕ **휴식 + 긴 휴식**: "몇 회 공부 후 휴식", 휴식 길이, 그리고 "몇 번째 휴식마다 긴 휴식"까지 설정
- 🔊 **완료음**: 삐삐/맑은 종/차임 3종 + **크기 조절**, 진동(지원 기기)
- 🔔 **잠금 화면 알림**: 아이폰 앱(ios-app)으로 설치하면 화면이 잠겨 있어도 종료 알림이 울림
- 📊 **진행**: 누적 공부시간 / 100시간까지 남은 시간 / 진행률 / 오늘 공부시간
- 📈 **통계**: 최근 7일 그래프, 월간·전체 통계, 연속 공부일(streak)
- 📅 **달력**: 하루 총 공부시간 표시 + 날짜별 **성찰 메모**
- 💡 **화면 켜두기**: 타이머를 보는 동안 화면이 꺼지지 않음(아이폰/아이패드 지원)
- 💾 **백업**: 기록을 파일로 내보내기/가져오기(기기를 바꿔도 이동 가능)

## PC에서 실행(테스트)
저장소 루트에서 아래를 실행한 뒤 브라우저로 접속합니다.

```bash
python3 -m http.server 8000
```
→ 브라우저에서 `http://localhost:8000` 열기.

> `index.html`을 파일로 바로 열어도 대부분 동작하지만, 오프라인(서비스워커) 기능은
> `http://` 주소로 열어야 켜집니다.

## 인터넷에 배포(GitHub Pages)
이 저장소에는 자동 배포 설정(`.github/workflows/deploy-pages.yml`)이 들어 있습니다.
`main` 브랜치가 바뀌면 자동으로 다음 주소에 배포됩니다:

```
https://dream0grow.github.io/100study/
```

**처음 한 번만** 저장소에서 아래를 설정해야 할 수 있습니다:
- GitHub 저장소 → **Settings → Pages → Build and deployment → Source = "GitHub Actions"**

설정 후 워크플로가 성공하면 위 주소로 아이폰 사파리에서 바로 열어 설치할 수 있습니다.

## 아이폰/아이패드에 설치

**방법 1 — 홈 화면에 추가 (간단):**
1. 위 GitHub Pages 주소를 사파리에서 엽니다.
2. 사파리 아래 **공유 버튼** → **"홈 화면에 추가"**.
3. 홈 화면 아이콘으로 실행하면 앱처럼 전체화면으로 열립니다.

**방법 2 — 진짜 아이폰 앱 (잠금 화면 알림 지원):**
맥에서 `ios-app/README.md`의 단계별 가이드를 따라 Xcode로 설치합니다.

### 화면 켜두기·소리 참고
- 화면 켜두기는 사파리 16.4 이상에서 동작합니다(설정에서 "화면 켜두기" 켜짐 상태).
- 홈으로 나갔다가 다시 앱으로 돌아오면 자동으로 다시 켜집니다.
- 아이폰 **무음 스위치**가 켜져 있으면 완료음이 안 날 수 있습니다.
- 완료음은 **미디어 볼륨**(옆면 버튼)을 따르고, 알림음은 **벨소리 볼륨**을 따릅니다.

## 데이터 저장
- 브라우저 `localStorage`에 저장됩니다(키: `study.v1`). 인터넷으로 전송되지 않습니다.
- 같은 기기·같은 브라우저에서만 기록이 유지됩니다.
- 기기를 바꾸려면 **설정 → 내보내기**로 파일을 저장한 뒤, 새 기기에서 **가져오기** 하세요.

## 파일 구성
```
(저장소 루트)
├── index.html            화면 구조
├── css/style.css         디자인
├── js/store.js           데이터 저장(localStorage)
├── js/timer.js           뽀모도로 타이머(시각 기준 정확 계산)
├── js/wakelock.js        화면 켜두기
├── js/beep.js            완료음(종류·크기)
├── js/notify.js          잠금 화면 예약 알림(아이폰 앱 전용)
├── js/progress.js        진행 화면
├── js/calendar.js        달력 + 성찰 메모
├── js/settings.js        설정 + 백업
├── js/app.js             전체 연결 + 탭 전환
├── manifest.webmanifest  설치 정보
├── sw.js                 오프라인 실행
├── icons/                앱 아이콘
└── ios-app/              아이폰 네이티브 포장(Capacitor)
```
